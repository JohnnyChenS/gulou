# AI 教育图谱项目

## 项目概述

全人生阶段的学习、成长、能力培养、兴趣培养平台。以开源 Prompt 库为知识引擎，通过 AI Agent 为用户提供长期陪伴式的个性化成长指导。覆盖 0 岁到老年全年龄层，育儿是当前切入点，最终目标是回归个人成长和兴趣学习。

## 核心设计决策

- **三条主线**：认知心理 + 身体能力 + 父母支持（父母支持仅在家庭期）
- **双入口**：主线（阶段→必备技能）、副线（兴趣→跨阶段路径）
- **用户分层**：0-14岁面向家长（育儿指导），14-18岁同时面向家长（育儿指导）和学习者本人（青春期阶段），18岁后完全过渡到学习者本人
- **阶段划分**：14岁后按生活阶段（青春期、大学期、职场开始、职场发展、家庭期、中年期、老年期），0-18岁育儿指导内容归属家庭期/育儿指导
- **灵活目录结构**：0-18岁育儿指导保持结构化（认知与心理、身体能力），14岁后学习者目录按阶段特点灵活命名（如身份探索、职业发展、健康管理等）
- **运动归属**：0-14岁在身体能力目录，14-18岁在身体发育目录，18岁后运动归入兴趣目录，但中年期/老年期保留健康维护目录中的必需运动
- **Prompt 归属**：主线 Prompt 在 `stages/`，兴趣 Prompt 在 `interests/`（按学习路线组织，阶段目录通过链接引用）
- **分层治理**：核心内容专家审核，社区贡献标记"community-contributed"
- **模型无关**：Prompt 不引用特定 AI 功能
- **中文优先**，后续国际化
- **纯内容仓库**：Markdown + Prompt 文件，无代码工具
- **父母支持维度**：不仅关注孩子能力发展，也关注照料者的心理状态和育儿指导（集中在家庭期）

## 产品策略

### 核心问题

当前 Prompt 设计只能解决"点"的问题（单次交互给出一次性指导），无法承载"线"（长期学习曲线）。学习乐器、语言、运动，或育儿中的习惯培养、心理健康建设，都是漫长的过程，需要持续引导和动态调整。

### 解决方向：AI Agent 模式

不做传统教育 App（会与 Duolingo、育儿 App 等正面竞争，无差异化），而是构建 **AI 长期陪伴导师**：
- Prompt 库作为 Agent 的知识引擎
- Agent 根据用户的人生阶段、学习目标、当前进度，动态组装 Prompt 并主动引导
- 用户不需要理解 Prompt，只需要和 Agent 对话
- Agent 具备长期记忆，了解用户情况、进度、偏好，提供持续陪伴感

### 工具层设计：多个垂直小工具

底层共享同一个开源 Prompt 库，上层拆分为多个独立垂直工具，各自独立上线、验证需求、变现：
- 育儿工具（输入月龄 → 成长指导方案）
- 兴趣学习工具（选兴趣 → 跨阶段学习路径）
- 习惯养成工具（输入习惯 → 基于行为科学的培养计划）
- 职业发展工具（输入职业阶段 → 能力提升方案）
- 健康工具（输入年龄/目标 → 运动/营养/心理方案）

### 变现路径

- **短期**：开源迭代内容 + GitHub Sponsors / 爱发电
- **中期**：垂直小工具上线，免费限次 + 付费无限
- **长期**：培训/咨询/企业合作

## 目录结构

```
stages/<阶段>/_index.md                      # 阶段概述+能力清单+学习路径
stages/<阶段>/<主题目录>/                     # 按阶段特点命名的主题目录
stages/家庭期（25-45岁）/育儿指导/                       # 父母支持主线（含0-18岁育儿指导）
stages/家庭期（25-45岁）/育儿指导/<年龄>/认知与心理/     # 0-18岁认知发展
stages/家庭期（25-45岁）/育儿指导/<年龄>/身体能力/       # 0-18岁运动发育
interests/<兴趣名>/                          # 兴趣领域（独立学习路径）
interests/<兴趣名>/_index.md                 # 学习路线总览+阶段映射
interests/<兴趣名>/beginner.md               # 入门Prompt
interests/<兴趣名>/intermediate.md           # 进阶Prompt
interests/<兴趣名>/advanced.md               # 高阶Prompt
references/                                  # 权威文献库
```

**阶段划分：**
- 14岁后（面向学习者本人）：青春期（14-18）、大学期（18-22）、职场开始（22-28）、职场发展（28-40）、家庭期（25-45）、中年期（40-60）、老年期（60+）
- 0-18岁（面向家长）：归属家庭期/育儿指导/，按年龄分：0-3、3-6、6-9、9-12、12-14、14-18

## Prompt 文件格式

每个 Prompt 文件必须包含：
- frontmatter：id、stage、track、domain、topic、age_range、difficulty、review_status、references、tags、learning_paths、related_prompts
- 双模式 Prompt：单轮模式（一次性输出）+ 多轮互动模式（对话式引导）
- 占位符 `[填写：xxx]` 供用户个性化
- "今天就能做的一件事"：具体到动作级别的即时行动
- 延伸探索：关联其他 Prompt

学习路径文件存放在 `paths/` 目录，定义 Prompt 之间的先后关系、完成标准和触发条件，是 Agent 实现长期陪伴引导的基础。

完整规范见 README.md。

## 提交规范

- Commit：`<type>(<scope>): <description>`
- 类型：content、index、refs、roadmap、docs、fix、i18n
- scope 为年龄阶段或目录名
- 一个 PR 做一件事

## 理论依据优先级

能力划分基于成熟发展心理学理论：皮亚杰认知发展、埃里克森心理社会发展、蒙台梭利敏感期、Bowlby依恋理论、Gallahue动作发展模型、LTAD长期运动员发展模型等。所有内容须有权威引用，禁止凭空编造。
