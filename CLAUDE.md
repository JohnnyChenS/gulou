# AI 教育图谱项目

## 项目概述

开源教育项目，按人生阶段组织能力培养路径，核心交付物为结构化 Prompt，用户可复制到任何 AI 模型获得个性化教育内容和指导。

## 核心设计决策

- **三条主线**：认知心理 + 身体能力 + 父母支持（父母支持仅在家庭期）
- **双入口**：主线（阶段→必备技能）、副线（兴趣→跨阶段路径）
- **用户分层**：0-14岁面向家长（育儿指导），14岁后过渡到学习者本人
- **阶段划分**：14岁后按生活阶段（青春期、大学期、职场开始、职场发展、家庭期、中年期、老年期），0-14岁内容归属家庭期/育儿指导
- **灵活目录结构**：0-14岁保持结构化（认知与心理、身体能力），14岁后按阶段特点灵活命名（如身份探索、职业发展、健康管理等）
- **运动归属**：0-14岁在身体能力目录，14-18岁在身体发育目录，18岁后运动归入兴趣目录，但中年期/老年期保留健康维护目录中的必需运动
- **Prompt 归属阶段**：`interests/` 仅存索引，实际 Prompt 在 `stages/` 中
- **分层治理**：核心内容专家审核，社区贡献标记"community-contributed"
- **模型无关**：Prompt 不引用特定 AI 功能
- **中文优先**，后续国际化
- **纯内容仓库**：Markdown + Prompt 文件，无代码工具
- **父母支持维度**：不仅关注孩子能力发展，也关注照料者的心理状态和育儿指导（集中在家庭期）

## 目录结构

```
stages/<阶段>/_index.md                      # 阶段概述+能力清单+学习路径
stages/<阶段>/<主题目录>/                     # 按阶段特点命名的主题目录
stages/<阶段>/兴趣/                           # 兴趣探索内容
stages/家庭期（25-45岁）/育儿指导/                       # 父母支持主线（含0-14岁育儿指导）
stages/家庭期（25-45岁）/育儿指导/<年龄>/认知与心理/     # 0-14岁认知发展
stages/家庭期（25-45岁）/育儿指导/<年龄>/身体能力/       # 0-14岁运动发育
interests/<兴趣名>/_index.md                 # 跨年龄索引
references/                                  # 权威文献库
```

**阶段划分：**
- 14岁后（面向学习者本人）：青春期（14-18）、大学期（18-22）、职场开始（22-28）、职场发展（28-40）、家庭期（25-45）、中年期（40-60）、老年期（60+）
- 0-14岁（面向家长）：归属家庭期/育儿指导/，按年龄分：0-3、3-6、6-9、9-12、12-14

## Prompt 文件格式

每个 Prompt 文件必须包含：
- frontmatter：id、stage、track、domain、topic、age_range、difficulty、review_status、references、tags
- 双模式 Prompt：单轮模式（一次性输出）+ 多轮互动模式（对话式引导）
- 占位符 `[填写：xxx]` 供用户个性化
- 延伸探索：关联其他 Prompt

完整规范见 README.md。

## 提交规范

- Commit：`<type>(<scope>): <description>`
- 类型：content、index、refs、roadmap、docs、fix、i18n
- scope 为年龄阶段或目录名
- 一个 PR 做一件事

## 理论依据优先级

能力划分基于成熟发展心理学理论：皮亚杰认知发展、埃里克森心理社会发展、蒙台梭利敏感期、Bowlby依恋理论、Gallahue动作发展模型、LTAD长期运动员发展模型等。所有内容须有权威引用，禁止凭空编造。
