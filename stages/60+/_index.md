# 长者期（60岁以上）

## 阶段概述

60岁以上是智慧结晶与身心安顿的阶段。核心任务是在维护认知功能和身体独立性的同时，完成生命叙事的整合，传承人生经验，保持社会连接。

## 认知与心理主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 智慧结晶 | 生活经验的高度整合 | 60+岁 |
| 选择性优化与补偿 | 聚焦重要之事，用策略弥补衰退 | 60+岁 |
| 认知衰退预防 | 神经可塑性训练 | 60+岁 |
| 记忆策略 | 外部辅助、编码策略 | 60+岁 |
| 生命叙事整合 | 接纳过去，找到意义 | 60+岁 |
| 社会连接维护 | 对抗孤独 | 60+岁 |
| 自主性维护 | 保持生活自主和决策能力 | 60+岁 |

## 身体能力主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 防跌倒训练 | 平衡、力量、反应速度 | 60+岁 |
| 骨密度维护 | 承重运动+抗阻训练 | 60+岁 |
| 关节活动度与柔韧性 | 维持日常活动范围 | 60+岁 |
| 温和有氧运动 | 散步、太极、水中运动 | 60+岁 |
| 认知-运动双重任务训练 | 运动同时进行认知挑战 | 60+岁 |
| 慢性病运动处方 | 糖尿病、高血压、关节炎 | 60+岁 |
| 功能独立性维护 | 日常活动能力 | 60+岁 |
