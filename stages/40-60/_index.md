# 中年期（40-60岁）

## 阶段概述

40-60岁是晶体智力达到峰值、智慧积累最丰富的时期，同时面临肌少症、骨密度下降、心血管风险上升等身体挑战。核心任务是在维持身心健康的同时，完成知识传承和意义重塑。

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 晶体智力峰值 | 知识、经验、判断力最强时期 | 40-60岁 |
| 模式识别专长 | 跨领域经验迁移 | 40-60岁 |
| 智慧与判断力 | 从知识到智慧 | 40-60岁 |
| 知识传承 | mentor/教练角色 | 40-60岁 |
| 生命回顾倾向 | 回顾和评价人生 | 40-60岁 |
| 中年转型与意义重塑 | 重新定义人生目标 | 40-60岁 |
| 情绪调节成熟 | 情绪管理能力达到高峰 | 40-60岁 |

## 身体能力主线
