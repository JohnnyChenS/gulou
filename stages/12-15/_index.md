# 青春期前期（12-15岁）

## 阶段概述

12-15岁是青春期启动和身份探索的关键期。认知上进入形式运算阶段，但情绪系统（杏仁核）先于认知控制系统（前额叶）成熟，导致情绪波动大、风险感知不成熟。身体上经历身高突增期，是力量和速度可训练性的关键窗口。

## 认知与心理主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 形式运算思维 | 假设-演绎推理、系统思考 | 12-15岁 |
| 身份探索 | 我是谁？我属于哪个群体？ | 12-15岁 |
| 情绪强度与波动 | 杏仁核先行，前额叶滞后 | 12-15岁 |
| 同伴影响峰值 | 从众压力最大 | 12-15岁 |
| 浪漫关系萌芽 | 对亲密关系的初步探索 | 12-15岁 |
| 风险感知不成熟 | 冲动与风险评估的神经发育错位 | 12-15岁 |
| 道德推理进阶 | 从规则服从到原则思考 | 12-15岁 |

## 身体能力主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| PHV身高突增期 | 生长速度峰值（女孩约11-13岁，男孩约13-15岁） | 12-15岁 |
| 力量与速度训练窗口 | 激素驱动可训练性提升 | 12-15岁 |
| 动作协调暂时性失调 | 肢体增长速率不匹配 | 12-15岁 |
| 运动损伤高发 | 生长板脆弱、过度使用 | 12-15岁 |
| 体像关注 | 对身体变化的关注和评估 | 12-15岁 |

## 理论依据

- 皮亚杰形式运算期（12+）
- 埃里克森身份vs角色混乱（12-18）
- Steinberg双系统模型（社会情感系统vs认知控制系统）
- Kohlberg道德发展习俗水平
- Marcia身份状态理论
- PHV模型（Mirwald等）
- 青春期力量/速度/耐力可训练窗口（Balyi & Hamilton）
- LTAD训练训练阶段（Train to Train）
- 青少年运动损伤预防（运动医学共识）
