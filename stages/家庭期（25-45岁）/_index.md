# 家庭期（25-45岁）

## 阶段概述

家庭期是人生中建立和经营家庭的关键阶段。本阶段的核心内容是**育儿指导**——为新手父母提供系统化的育儿指导和心理支持，覆盖 0-18 岁全年龄段。

> **新手爸妈？** 如果你是第一次当爸妈，不知道从哪里开始，先看这里：[新手爸妈快速入口](育儿指导/quick-start.md) — 按场景组织，找到你现在最需要的问题就好。

## 学习路径

> 详细的学习路径定义见 `paths/` 目录。以下是简化版依赖关系。

### 父母支持路径

```
产前准备（孕晚期-出生）
  └→ 新手父母心理调适（0-12月）
       ├→ 睡眠剥夺应对（0-12月）
       ├→ 产后抑郁识别（0-24月）
       └→ 夫妻关系调适（0-36月）
            └→ 代际育儿冲突（0-6岁）
                 └→ 育儿社交支持（0-6岁）
```

**并行模块**（无前置依赖，任何阶段可开始）：
- 育儿焦虑管理（0-18岁）
- 比较与竞争心理（0-18岁）
- 完美主义陷阱（0-18岁）
- 兴趣与身份保持（0-18岁）
- 父母身心健康（0-18岁）
- 父母自我时间管理（0-18岁）
