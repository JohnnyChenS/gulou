# 婴幼儿期（0-3岁）

## 阶段概述

0-3岁是人生发展最迅速的时期，大脑神经连接以每秒百万级的速度形成。此阶段的核心任务是建立对世界的基本信任，发展感知运动能力，为后续所有发展奠定基础。

对孩子来说是如此，对父母来说同样如此——成为父母是人生中最深刻的角色转变之一。本阶段不仅包含孩子的能力发展主线，也包含**父母支持主线**，帮助新手父母平稳度过这段充满挑战与美好的时光。

> **新手爸妈？** 如果你是第一次当爸妈，不知道从哪里开始看，先看这里：[新手爸妈快速入口](../quick-start.md) — 按场景组织，找到你现在最需要的那个问题就好。

## 学习路径图

以下展示各主题之间的**前置依赖关系**。箭头表示"先学 A，再学 B"或"A 是 B 的认知基础"。

```
安全依恋（0-18月）——所有心理发展的基石
  ├→ 因果关系推理（4-12月）
  │    └→ 联合注意（9-15月）
  ├→ 客体永久性（6-18月）
  │    └→ 符号思维萌芽（12-24月）
  └→ 自我意识觉醒（15-24月）
       ├→ 共情萌芽（18-36月）
       └→ 自主意志（18-36月）
```

**文字版依赖关系：**

1. **安全依恋**（0-18月）— 所有心理发展的基石，没有安全依恋就没有探索的勇气
2. **客体永久性**（6-18月）— 符号思维的前提：先理解"看不见的东西存在"，才能用一物代表另一物
3. **因果关系推理**（4-12月）— 与客体永久性互相促进：理解"我的动作导致物品消失/出现"
4. **联合注意**（9-15月）— 因果推理的社会延伸：从"我的动作影响物体"到"我的行为引起他人注意"
5. **自我意识觉醒**（15-24月）— 共情和自主意志的认知基础：先区分"我的感受"和"你的感受"
6. **符号思维萌芽**（12-24月）— 依赖客体永久性和自我意识
7. **共情萌芽**（18-36月）— 依赖安全依恋和自我意识：先被理解，才能理解他人
8. **自主意志**（18-36月）— 依赖自我意识：从"我是独立的个体"到"我可以自己做决定"

## 理论依据

- 皮亚杰感知运动期（0-2）
- 埃里克森信任vs怀疑（0-1.5）、自主vs羞怯（1.5-3）
- Bowlby依恋理论
- Lewis镜像自我识别研究（15-24月）
- 蒙台梭利语言/感官/动作敏感期
- WHO运动发育里程碑标准
- Gallahue粗大动作发展模型
- Ayres感觉统合理论
