# 0-3 岁发展路线图与评估框架

## 如何使用这份工具

这份评估框架帮助你了解孩子在 0-3 岁各核心能力上的发展水平，识别需要关注的领域，并提供可执行的下一步行动。

**使用频率建议**：
- 0-12 个月：每 2 个月做一次评估（发展变化快）
- 12-36 个月：每 3 个月做一次评估
- 日常观察随时记录（看到相关行为就勾选）

**重要提醒**：0-3 岁是发展差异最大的阶段——同一个能力，正常发展的范围可能跨越 3-6 个月。评估结果是参考而非标签。如果某项能力持续落后同龄 6 个月以上，建议咨询儿科医生或发育行为专家。

## 二、观察清单（勾选式）

以下每项能力列出可观察的行为。勾选孩子**已经能做到**的项目。

**计分方式**：每项 1 分。参考标准根据孩子月龄调整——越接近该能力阶段上限的月龄，期望勾选越多。

### 语言发展

- [ ] 能区分不同声音，对人声有偏好（0-3 个月）
- [ ] 能发出元音声（a、o、u）和咕咕声（cooing）（2-4 个月）
- [ ] 能发出辅音+元音组合（"ba""ma""da"）（4-6 个月）
- [ ] 对自己的名字有反应（6-9 个月）
- [ ] 能理解常见词（"不""抱抱""奶瓶"）（9-12 个月）
- [ ] 出现第一个有意义的词（如"妈妈""爸爸"）（9-15 个月）
- [ ] 能用手指向想要的东西（12-15 个月）
- [ ] 词汇量达到 20-50 个词（15-18 个月）
- [ ] 能说两个词的组合（"妈妈抱""要饼干"）（18-24 个月）
- [ ] 词汇量快速增长，每周 8-10 个新词（18-24 个月）
- [ ] 能说简单句（"我要喝水""爸爸去上班"）（21-24 个月）
- [ ] 开始使用"为什么""什么"提问（24-30 个月）
- [ ] 能用复杂句表达（使用"和""但是""因为"）（30-36 个月）
- [ ] 能讲述过去发生的事（30-36 个月）

### 安全依恋

- [ ] 听到照料者的声音会安静下来或转头寻找（0-3 个月）
- [ ] 对照料者出现社会性微笑（2-3 个月）
- [ ] 被陌生人抱起时会不安或哭闹（6-10 个月）
- [ ] 照料者离开时会哭闹（分离焦虑，8-12 个月）
- [ ] 照料者回来时会伸手要抱、很快平复（8-12 个月）
- [ ] 以照料者为安全基地探索新环境（12-18 个月）
- [ ] 受伤或害怕时会主动回到照料者身边寻求安慰（12-24 个月）
- [ ] 能在照料者在场的情况下和陌生人互动（18-36 个月）

### 因果关系

- [ ] 发现自己的动作能产生结果（踢腿→摇铃响）（0-4 个月）
- [ ] 有意重复有趣的结果（反复拍打玩具让它响）（4-8 个月）
- [ ] 使用工具达到目的（拉布条够到远处的玩具）（8-12 个月）
- [ ] 理解简单的日常因果（按开关→灯亮）（12-18 个月）
- [ ] 能用语言表达因果关系（"因为...所以..."）（24-36 个月）

### 客体永久性

- [ ] 对部分遮挡的物品有反应（6-8 个月）
- [ ] 能寻找完全遮挡的物品（8-12 个月）
- [ ] 能寻找被多次转移的物品（12-18 个月）
- [ ] 喜欢躲猫猫游戏（任何月龄出现都可以）

### 联合注意

- [ ] 能跟随大人的目光方向（3-6 个月）
- [ ] 能跟随大人的手指方向看东西（9-12 个月）
- [ ] 能用手指向自己想要的东西（12-15 个月）
- [ ] 能用手指向有趣的东西和大人分享（"妈妈看！"）（15-18 个月）
- [ ] 能参与简单的共同活动（一起搭积木、一起看绘本）（18-36 个月）

### 自主性

- [ ] 开始对某些事情说"不"（12-18 个月）
- [ ] 坚持自己做某事（自己吃饭、自己穿鞋）（18-24 个月）
- [ ] 能在两个选项中做选择（"要苹果还是香蕉？"）（24-36 个月）
- [ ] 开始理解简单的规则和边界（30-36 个月）
- [ ] 在被拒绝时能接受替代方案（"不能吃糖，但可以吃饼干"）（24-36 个月）

### 自我意识

- [ ] 在镜子中认出自己（用手指镜子中的自己而非以为是另一个宝宝）（12-18 个月）
- [ ] 知道自己的名字（被叫名字会回应）（12-18 个月）
- [ ] 开始说"我的"（18-24 个月）
- [ ] 知道自己的性别（24-30 个月）
- [ ] 能用 2-3 个特征描述自己（"我叫XX，我X 岁"）（30-36 个月）
- [ ] 出现自豪、尴尬等自我意识情绪（18-36 个月）

### 符号思维（假装游戏）

- [ ] 出现功能性假装（假装喝水、假装打电话）（12-18 个月）
- [ ] 能用物品代表其他物品（用积木当手机）（18-24 个月）
- [ ] 能进行有角色的假装游戏（当医生、当妈妈）（24-36 个月）
- [ ] 能进行有情节的假装游戏（有开头、经过、结果）（30-36 个月）
- [ ] 喜欢"过家家"类游戏（24-36 个月）

### 共情

- [ ] 看到别人哭会注视或跟着哭（6-12 个月）
- [ ] 会尝试安慰哭泣的人（拍拍、递东西）（12-18 个月）
- [ ] 能区分自己和他人的情绪（"他难过"vs"我难过"）（18-24 个月）
- [ ] 会主动分享自己的东西给需要的人（24-36 个月）
- [ ] 理解别人的情绪可能和自己不同（"他喜欢这个，我不喜欢"）（30-36 个月）

### 粗大动作

- [ ] 俯卧抬头 45°（2-3 个月）
- [ ] 翻身（4-6 个月）
- [ ] 独坐稳（6-8 个月）
- [ ] 爬行（7-10 个月）
- [ ] 扶站（8-10 个月）
- [ ] 扶走（9-12 个月）
- [ ] 独立行走（10-15 个月）
- [ ] 能跑（18-24 个月）
- [ ] 能双脚跳（24-36 个月）
- [ ] 能上下楼梯交替步（30-36 个月）

### 精细动作

- [ ] 全掌抓握（3-5 个月）
- [ ] 传递物品从一只手到另一只（5-7 个月）
- [ ] 钳形抓握（拇指+食指捏取小物）（8-12 个月）
- [ ] 有意识释放物品（能把东西放进容器）（10-14 个月）
- [ ] 叠 2-3 块积木（12-18 个月）
- [ ] 翻书页（18-24 个月）
- [ ] 叠 6-8 块积木（24-30 个月）
- [ ] 用蜡笔画闭合的圆形（30-36 个月）
- [ ] 能用勺子自己吃饭（基本不洒）（24-36 个月）

### 感觉统合

- [ ] 对声音有定位反应（转头寻找声源）（3-6 个月）
- [ ] 能追踪移动物体 180°（4-6 个月）
- [ ] 对不同质地有明显偏好（喜欢/抗拒某些触感）（6-12 个月）
- [ ] 能在不同地面行走（草地、沙地、不平整地面）（12-24 个月）
- [ ] 能协调多种感觉输入（边看边听边动）（18-36 个月）
- [ ] 不过度抗拒日常感觉刺激（如洗头、剪指甲）（12-36 个月）

### 口部运动

- [ ] 吸吮吞咽协调（0-3 个月）
- [ ] 能接受辅食（4-6 个月）
- [ ] 能咀嚼软食（6-9 个月）
- [ ] 能用杯子喝水（12-18 个月）
- [ ] 能清晰地说出常见词汇（24-36 个月）
- [ ] 能用吸管喝水（12-18 个月）

## 三、场景测试

以下场景在日常中自然观察即可，不需要刻意"测试"孩子。

### 语言发展：命名游戏

**场景**：指着一个孩子熟悉的物品（如球、杯子、灯），问"这是什么？"

**观察**：
- ✅ 能正确说出物品名称（18 个月以上）→ 语言发展良好
- ⚠️ 能理解问题但说不出名字，或用"这个"代替 → 正在发展中
- ❌ 对问题没有反应，不尝试表达 → 可以多做命名游戏，每天指着物品说名字

### 安全依恋：陌生情境（简化版）

**场景**：带孩子去一个不太熟悉的地方（如朋友家），照料者在场时观察孩子的行为。

**观察**：
- ✅ 能以照料者为"安全基地"探索环境，时不时回头看 → 安全依恋
- ⚠️ 紧贴照料者不敢探索，或完全不理会照料者自顾自玩 → 需要关注依恋模式
- ❌ 对照料者的离开和回来都没有明显反应 → 建议咨询专业人士

### 客体永久性：藏玩具

**场景**：当着孩子面把一个玩具藏在毯子下。

**观察**：
- ✅ 会掀开毯子找玩具（8 个月以上）→ 客体永久性发展良好
- ⚠️ 看着毯子但不去找 → 正在发展中
- ❌ 以为玩具消失了（不看毯子方向）→ 可以多玩躲猫猫游戏

### 联合注意：指物分享

**场景**：看到一个有趣的东西（如窗外的小鸟），对孩子说"看！小鸟！"并用手指。

**观察**：
- ✅ 会顺着手指方向看过去，然后回头看你（9 个月以上）→ 联合注意发展良好
- ⚠️ 会看你或看你的手指，但不会看手指指的方向 → 正在发展中
- ❌ 完全不关注你的指物行为 → 可以多做"指东西给孩子看"的互动

### 自主性：有限选择

**场景**：给孩子两个选项（如"穿红色鞋子还是蓝色鞋子？"）。

**观察**：
- ✅ 能做出选择并表达（18 个月以上）→ 自主性发展良好
- ⚠️ 犹豫不决，需要你帮忙决定 → 正在发展中
- ❌ 不理解选择的含义，或对两个选项都说"不" → 可以从更简单的选择开始

### 自我意识：镜子测试

**场景**：趁孩子不注意，在他鼻子上贴一个小贴纸，然后让他照镜子。

**观察**：
- ✅ 去摸自己鼻子上的贴纸（而非镜子里的）（15 个月以上）→ 自我意识发展良好
- ⚠️ 去摸镜子 → 正在发展中
- ❌ 对镜子中的自己没有特别反应 → 可以多玩镜子游戏

### 符号思维：假装游戏

**场景**：给孩子一个积木和一个空杯子，观察他怎么玩。

**观察**：
- ✅ 会假装用杯子喝水，或假装给娃娃喂水（18 个月以上）→ 符号思维发展良好
- ⚠️ 只是把积木放进杯子（功能性操作）→ 正在发展中
- ❌ 只是拿着积木敲打或扔掉 → 可以多示范假装游戏

### 粗大动作：行走和攀爬

**场景**：在安全的户外环境，观察孩子如何移动。

**观察**：
- ✅ 能独立行走、蹲下捡东西、站起来继续走（15 个月以上）→ 粗大动作发展良好
- ⚠️ 能走但经常摔跤、不敢蹲下 → 正在发展中
- ❌ 15 个月仍不能独立行走 → 建议咨询儿科医生

如果你有 AI 工具（如 ChatGPT、Claude），可以将以下内容发送给它，获得个性化评估分析：

> 你是一位儿童发展专家，擅长帮助家长评估 0-3 岁婴幼儿的发展水平。请根据我提供的观察结果，分析我孩子的发展状况并给出建议。
>
> **孩子信息**：
> - 月龄：[填写：如"8 个月"/"1 岁 4 个月"]
> - 性别：[填写]
> - 出生情况：[填写：如"足月顺产"/"早产 3 周"/"剖腹产"，如果觉得相关的话]
>
> **观察清单勾选结果**（我勾选了孩子能做到的项目）：
>
> 语言发展（共 14 项）：[填写勾选了几项，或列出具体勾选了哪些]
> 安全依恋（共 8 项）：[填写勾选了几项，或列出具体勾选了哪些]
> 因果关系（共 5 项）：[填写]
> 客体永久性（共 4 项）：[填写]
> 联合注意（共 5 项）：[填写]
> 自主性（共 5 项）：[填写]
> 自我意识（共 6 项）：[填写]
> 符号思维（共 5 项）：[填写]
> 共情（共 5 项）：[填写]
> 粗大动作（共 10 项）：[填写]
> 精细动作（共 9 项）：[填写]
> 感觉统合（共 6 项）：[填写]
> 口部运动（共 6 项）：[填写]
>
> **场景测试结果**（可选）：
> [填写你在场景测试中观察到的情况]
>
> **我最关心的问题**：
> [填写：如"孩子 10 个月了还不会爬""分离焦虑很严重""好像对什么都不感兴趣"]
>
> **请按以下格式输出**：
>
> 1. **发展水平评估**：基于我的描述和孩子月龄，判断各能力域的发展水平（超前/正常/需要关注），标注亮点和需要关注的领域
> 2. **与同龄水平的对比**：基于 0-3 岁发展心理学研究，我的孩子在哪些方面处于同龄平均水平？哪些方面可能偏高或偏低？（注意：0-3 岁正常发展范围很宽，不要轻易判定"落后"）
> 3. **优先行动计划**：建议我优先关注哪 1-2 个能力？为什么？
> 4. **具体行动建议**：针对需要关注的能力，给出 3 个具体的、今天就能做的亲子互动（具体到"做什么、怎么做、做多久"）
> 5. **下次评估建议**：2-3 个月后再次评估时，我应该重点观察哪些变化？
> 6. **是否需要专业评估**：基于我的描述，是否建议咨询儿科医生或发育行为专家？给出判断依据
>
> 请基于发展心理学研究给出专业分析。0-3 岁的发展差异很大，请避免制造焦虑，但如果确实有需要关注的信号，请明确指出。
