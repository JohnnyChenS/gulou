# 12-14 岁发展路线图与评估框架

## 如何使用这份工具

这份评估框架帮助你了解 12-14 岁孩子的核心发展任务，识别需要关注的领域。

**使用频率建议**：每 6 个月做一次评估

**重要提醒**：12-14 岁是青春期的起始阶段，孩子开始经历快速的生理和心理变化。评估结果是参考而非标签。

---

## 一、发展路线图

### 12-13 岁：青春期前期

| 能力域 | 预期表现 | 深入了解 |
|--------|---------|---------|
| 亲子沟通 | 开始嫌父母烦；更愿意和朋友聊天 | [青春期前期沟通](认知与心理/puberty-communication-01.md) |
| 同伴关系 | 从"玩伴"到"知己"；同伴压力升级 | [同伴关系深化](认知与心理/peer-relationship-deep-01.md) |
| 身体发育 | 身高突增；第二性征开始发育 | [身体发育与性教育](认知与心理/body-development-01.md) |
| 学业压力 | 初中适应；课程难度增加 | [学业压力与时间管理](认知与心理/academic-time-management-01.md) |
| 网络素养 | 开始独立使用社交媒体 | [网络素养启蒙](认知与心理/digital-literacy-01.md) |
| 自我认同 | 开始思考"我是谁" | [自我认同与价值观](认知与心理/identity-exploration-01.md) |

### 14 岁：过渡期

| 能力域 | 预期表现 |
|--------|---------|
| 亲子沟通 | 沟通模式基本转型 |
| 同伴关系 | 同伴影响力达到峰值 |
| 身体发育 | 身体变化加速 |
| 学业压力 | 学业压力增大 |
| 网络素养 | 网络使用更加独立 |

## 二、观察清单（勾选式）

### 亲子沟通

- [ ] 能和父母进行基本对话
- [ ] 愿意分享学校的事
- [ ] 能接受父母的建议
- [ ] 能表达自己的观点
- [ ] 能在冲突中保持冷静
- [ ] 愿意和父母讨论重要话题

### 同伴关系

- [ ] 能建立稳定友谊
- [ ] 能处理同伴压力
- [ ] 能区分"真正的朋友"和"利用你的人"
- [ ] 能解决同伴冲突
- [ ] 能在群体中找到自己的位置
- [ ] 理解"受欢迎"和"被喜欢"的区别

### 身体发育

- [ ] 了解青春期身体变化的正常范围
- [ ] 能接受自己身体的变化
- [ ] 了解基本的性知识
- [ ] 能区分"正常"和"需要关注"的身体变化
- [ ] 能和父母讨论身体变化
- [ ] 了解身体隐私和边界

### 学业压力

- [ ] 能适应初中学习节奏
- [ ] 能完成作业
- [ ] 能管理学习时间
- [ ] 能应对考试压力
- [ ] 能区分"努力"和"焦虑"
- [ ] 能在学业和休息之间平衡

### 网络素养

- [ ] 能识别基本的网络风险
- [ ] 能保护个人隐私
- [ ] 能区分真假信息
- [ ] 能管理屏幕时间
- [ ] 能应对网络欺凌
- [ ] 能和父母讨论网络使用

### 自我认同

- [ ] 开始思考"我是谁"
- [ ] 能探索不同兴趣
- [ ] 能接受自己和父母不同
- [ ] 能表达自己的价值观
- [ ] 能接受"我不确定"
- [ ] 能在探索中保持安全
