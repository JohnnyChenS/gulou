# 14-18 岁发展路线图与评估框架

## 如何使用这份工具

这份评估框架帮助你了解 14-18 岁孩子的核心发展任务，识别需要关注的领域。

**使用频率建议**：每 6 个月做一次评估

**重要提醒**：14-18 岁是青春期的核心阶段，孩子正在从"儿童"向"成人"过渡。评估结果是参考而非标签。

---

## 一、发展路线图

### 14-16 岁：青春期核心期

| 能力域 | 预期表现 | 深入了解 |
|--------|---------|---------|
| 亲子沟通 | 从"命令-服从"转向"对话-协商" | [青春期亲子沟通](认知与心理/parent-teen-communication-01.md) |
| 叛逆与冲突 | 追求自主；挑战规则 | [叛逆与冲突应对](认知与心理/adolescent-rebellion-01.md) |
| 心理健康 | 情绪波动；身份探索 | [心理健康危机识别](认知与心理/adolescent-mental-health-01.md) |
| 升学压力 | 高考焦虑；学业压力 | [升学压力与学业焦虑](认知与心理/academic-pressure-01.md) |

| 能力域 | 预期表现 |
|--------|---------|
| 亲子沟通 | 沟通模式基本稳定 |
| 叛逆 | 叛逆行为减少 |
| 心理健康 | 身份认同逐渐稳定 |

| 数字生活 | 数字素养基本建立 |

## 二、观察清单（勾选式）

### 亲子沟通

- [ ] 能和父母进行基本对话
- [ ] 愿意分享学校的事
- [ ] 能接受父母的建议
- [ ] 能表达自己的观点
- [ ] 能在冲突中保持冷静
- [ ] 愿意和父母讨论重要话题
- [ ] 能在需要时主动寻求帮助

### 叛逆与冲突

- [ ] 能区分"可以放手"和"必须坚守"
- [ ] 能在冲突中保持冷静
- [ ] 能接受合理的规则
- [ ] 能表达自己的需求
- [ ] 能在冲突后修复关系
- [ ] 能理解父母的立场

### 心理健康

- [ ] 情绪基本稳定
- [ ] 能应对学业压力
- [ ] 能处理同伴关系
- [ ] 能接受自己的不完美
- [ ] 能在需要时寻求帮助
- [ ] 没有自伤或自杀想法

### 升学压力

- [ ] 能应对学业压力
- [ ] 能管理学习时间
- [ ] 能在考试中保持冷静
- [ ] 能区分"努力"和"焦虑"
- [ ] 能在学业和休息之间平衡
- [ ] 能接受"尽力就好"

### 数字生活

- [ ] 能管理屏幕时间
- [ ] 能识别网络风险
- [ ] 能保护个人隐私
- [ ] 能区分真假信息
- [ ] 能应对网络欺凌
- [ ] 能和父母讨论网络使用
