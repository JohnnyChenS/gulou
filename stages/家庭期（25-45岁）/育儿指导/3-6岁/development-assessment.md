# 3-6 岁发展路线图与评估框架

## 如何使用这份工具

这份评估框架帮助你了解孩子在 3-6 岁各核心能力上的发展水平，识别需要关注的领域，并提供可执行的下一步行动。

**使用频率建议**：
- 每 3 个月做一次完整评估（约 1 小时）
- 日常观察随时记录（看到相关行为就勾选）
- 每次评估后对比上次结果，观察进步趋势

**评估结果不是"诊断"**：每个孩子的发展节奏不同，评估结果是参考而非标签。如果某项能力持续低于同龄水平，建议咨询专业人士。

## 二、观察清单（勾选式）

以下每项能力列出 6-8 个可观察的行为。勾选孩子**已经能做到**的项目，统计得分。

**计分方式**：每项 1 分。总分参考：
- 5-6 分：该能力发展良好
- 3-4 分：该能力正在发展中，属正常范围
- 0-2 分：建议关注，可重点练习相关主题中的活动

### 心理理论

- [ ] 能理解别人和自己有不同的喜好（"妈妈喜欢吃辣，我不喜欢"）
- [ ] 能理解别人和自己有不同的想法（"爸爸不知道我藏在哪里"）
- [ ] 能区分"假装"和"真实"（"这是假装的蛋糕，不能吃"）
- [ ] 能理解错误信念（小明以为球在盒子里，但其实被移走了）
- [ ] 能理解善意的谎言（"她是为了不让他难过才说喜欢的"）
- [ ] 会使用心理状态词汇（想、以为、觉得、希望、知道）
- [ ] 能预测别人的情绪反应（"他知道了一定会很高兴"）

### 执行功能

- [ ] 能遵守 2 步指令（"把杯子拿过来放在桌上"）
- [ ] 能在游戏规则中抑制冲动（如"红灯绿灯"游戏中红灯时能停住）
- [ ] 能记住并执行 3 个步骤的任务
- [ ] 能在任务之间切换（从画画切换到吃饭，不崩溃）
- [ ] 能按顺序回忆 3-4 个物品
- [ ] 能在干扰中保持注意力 10 分钟以上
- [ ] 能用简单策略解决问题（"我够不到，可以用椅子垫高"）

### 延迟满足

- [ ] 能在提醒下等待 2-3 分钟（如等妈妈打完电话再说）
- [ ] 能理解"先做完XX再做XX"的安排
- [ ] 能在诱惑面前使用策略（如唱歌、数数、想别的事情）
- [ ] 能为了更大的奖励选择等待（如"现在给 1 颗糖，等 10 分钟给 3 颗"）
- [ ] 能在游戏中遵守"轮到你了/轮到我了"的规则
- [ ] 能在被拒绝时不崩溃大哭（偶尔可以，不要求每次都做到）

### 情绪调节

- [ ] 能用语言表达自己的情绪（"我生气了""我害怕"）
- [ ] 能识别图片中人物的基本情绪
- [ ] 能在提醒下使用深呼吸/数数来平复情绪
- [ ] 能理解情绪的原因（"他哭了，因为他的玩具坏了"）
- [ ] 能在冲突后主动尝试修复关系（道歉、拥抱）
- [ ] 能接受"不能马上得到想要的东西"而不大发脾气
- [ ] 能用语言而非身体动作表达不满

### 社会性游戏

- [ ] 能和一个小朋友一起玩 10 分钟以上
- [ ] 能在游戏前商量"我们玩什么"
- [ ] 能在游戏中轮流和分享（在提醒下也可以）
- [ ] 能参与有简单规则的合作游戏
- [ ] 能在游戏中扮演角色（医生、厨师等）
- [ ] 能在冲突中尝试协商（"我先玩，然后给你"）
- [ ] 能加入别人正在进行的游戏

### 前阅读与前数学

- [ ] 喜欢听故事，能复述简单故事的主要情节
- [ ] 能认出自己的名字
- [ ] 能区分文字和图画
- [ ] 能从 1 数到 20
- [ ] 能理解数量对应（给你 3 颗糖，数出来确实是 3 颗）
- [ ] 能比较大小、长短、多少
- [ ] 能按颜色、形状、大小分类
- [ ] 能认出并写出部分字母或简单汉字

### 性别认同与自我概念

- [ ] 能正确说出自己的性别
- [ ] 能用 3 个以上特征描述自己（"我叫XX，我X 岁，我喜欢XX"）
- [ ] 理解性别不会因外表改变（"穿裙子的男孩还是男孩"）
- [ ] 能说出自己的优点（"我跑得很快""我会画画"）
- [ ] 对自己的尝试表现出自豪感
- [ ] 能接受"我还不太会，但我可以学"

### 基本动作技能

- [ ] 能平稳地跑步，不经常摔跤
- [ ] 能双脚起跳，落地稳定
- [ ] 能单脚跳 3 步以上
- [ ] 能接住从 2 米外抛来的大球
- [ ] 能向前踢球
- [ ] 能投掷物体到 3 米外的目标方向
- [ ] 能上下楼梯交替步

### 平衡与协调

- [ ] 能单脚站立 5 秒以上
- [ ] 能沿直线走 5 米不偏离
- [ ] 能倒退走 3 步以上
- [ ] 能在低矮的平衡木上走 2 米
- [ ] 能原地转 3 圈后不摔倒
- [ ] 能手脚并用爬上攀爬架

### 手眼协调

- [ ] 能用积木搭建 5 层以上的塔
- [ ] 能串大珠子（每分钟 5 颗以上）
- [ ] 能用剪刀沿直线剪
- [ ] 能画闭合的圆形和直线
- [ ] 能握笔画简单的画（人、房子）
- [ ] 能把球投进 1 米外的篮子里（10 次中 3 次以上）

### 利手确立

- [ ] 在拿笔、吃饭、扔球时表现出一致的用手偏好
- [ ] 用优势手完成精细操作（如扣扣子、用勺子）比非优势手明显更熟练
- [ ] 能用优势手稳定地握笔涂鸦

### 户外活动习惯

- [ ] 每天有 1 小时以上的户外活动时间
- [ ] 喜欢在户外跑跳、攀爬、探索
- [ ] 能在不同天气（晴天、阴天、小雨）下进行户外活动
- [ ] 每天屏幕时间不超过 1 小时

## 三、场景测试

以下场景用于在自然情境中观察孩子的能力发展水平。不需要"测试"孩子——在日常中遇到这些情境时，有意识地观察即可。

### 心理理论：猜猜我在想什么

**场景**：把一颗糖放进盒子 A，趁孩子不注意移到盒子 B。问孩子："如果爸爸现在进来找糖，他会去哪里找？"

**观察**：
- ✅ 说"盒子 A"（理解错误信念）→ 该能力发展良好
- ⚠️ 说"盒子 B"（从自己知道的位置推断）→ 正在发展中
- ❌ 不理解问题 → 可以多玩"猜猜他在想什么"的游戏

### 执行功能：三步指令

**场景**：对孩子说"请你去厨房，从柜子里拿一个苹果，然后放在客厅的桌子上"。

**观察**：
- ✅ 完成全部三步 → 执行功能发展良好
- ⚠️ 完成两步，忘记第三步 → 工作记忆正在发展
- ❌ 第一步就忘了后面的 → 可以从两步指令开始练习

### 延迟满足：棉花糖测试（简化版）

**场景**：给孩子一块饼干，说"我现在要去做一件事，如果你等我回来再吃，我会再给你一块。如果你现在就想吃，也可以，但就没有第二块了。"离开房间 2-3 分钟。

**观察**：
- ✅ 等到你回来 → 延迟满足能力良好
- ⚠️ 犹豫了一下但吃了 → 正在发展中
- ❌ 立刻吃了 → 可以从 30 秒的等待开始练习

### 情绪调节：被拒绝时的反应

**场景**：孩子想要一个东西（如玩具、零食），你温和但坚定地拒绝。

**观察**：
- ✅ 能用语言表达不满（"我很想要"），虽然难过但能接受 → 情绪调节良好
- ⚠️ 哭闹一会儿但在安抚下能平静 → 正在发展中
- ❌ 持续大哭大闹、打人、无法安抚 → 需要重点练习情绪调节策略

### 社会性游戏：和同伴的互动

**场景**：带孩子去有同龄小朋友的地方（公园、游乐场），观察 10 分钟。

**观察**：
- ✅ 能主动加入别人的游戏，或邀请别人一起玩 → 社交能力良好
- ⚠️ 在旁边观察一会儿后加入，或需要大人帮忙介绍 → 正在发展中
- ❌ 始终自己玩，不愿靠近其他孩子 → 可以从一对一的玩伴约会开始

### 前数学：分糖果

**场景**：拿出 6 颗糖，说"请把这些糖平均分给爸爸和妈妈"。

**观察**：
- ✅ 能一人分 3 颗，理解"一样多" → 数学概念良好
- ⚠️ 能分但不平均（如一人 4 一人 2）→ 正在发展中
- ❌ 不理解"分"的概念 → 可以从 2 颗糖分给 2 个人开始

### 基本动作：跳远和接球

**场景**：在户外，画一条线让孩子往前跳；从 2 米外抛大球给孩子。

**观察**：
- ✅ 能双脚起跳落地稳定，能接住大球 → 动作发展良好
- ⚠️ 能跳但落地不稳，能碰到球但接不住 → 正在发展中
- ❌ 不能双脚起跳，回避抛来的球 → 可以从原地跳和近距离抛接开始
