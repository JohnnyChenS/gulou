# 9-12 岁发展路线图与评估框架

## 如何使用这份工具

这份评估框架帮助你了解孩子在 9-12 岁各核心能力上的发展水平，识别需要关注的领域。

**使用频率建议**：每 6 个月做一次评估

---

## 一、发展路线图

### 9-10 岁：同伴关系深化期

| 能力域 | 预期表现 | 深入了解 |
|--------|---------|---------|
| 同伴关系 | 开始关注"受欢迎"程度；同伴压力开始显现 | [同伴关系](认知与心理/peer-relationships-01.md) |
| 历史观与世界观 | 能理解"过去"和"不同文化" | [历史观](认知与心理/historical-worldview-01.md) |
| 演讲与公众表达 | 能在班级面前做简短发言 | [演讲](认知与心理/public-speaking-01.md) |
| 创新思维 | 能想出多种解决方案 | [创新思维](认知与心理/innovative-thinking-01.md) |
| 消费决策 | 能理解广告策略；能做简单预算 | [消费决策](认知与心理/financial-decision-01.md) |
| 运动习惯 | 广泛尝试不同运动 | [运动习惯](身体能力/sports-habits-01.md) |
| 视力保护 | 近视防控关键期 | [视力保护](身体能力/vision-protection-01.md) |
| 睡眠管理 | 9-12 小时睡眠 | [睡眠管理](身体能力/sleep-management-01.md) |

### 11-12 岁：青春期准备期

| 能力域 | 预期表现 |
|--------|---------|
| 同伴关系 | 群体认同需求增强；能处理同伴压力 |
| 历史观 | 能理解因果关系和多元视角 |
| 演讲 | 能做 3-5 分钟有结构的演讲 |
| 创新思维 | 能把想法变成行动 |
| 消费决策 | 能管理一周的零花钱 |

## 二、观察清单（勾选式）

### 同伴关系

- [ ] 能和同龄人建立稳定友谊
- [ ] 能处理同伴压力
- [ ] 能区分"受欢迎"和"被喜欢"
- [ ] 能解决同伴冲突
- [ ] 能在群体中找到自己的位置
- [ ] 理解"真正的朋友"的标准

### 历史观与世界观

- [ ] 能理解"过去"的概念
- [ ] 能理解因果关系
- [ ] 能理解不同文化的存在
- [ ] 能区分事实和观点
- [ ] 对历史故事感兴趣
- [ ] 能从不同角度思考问题

### 演讲与公众表达

- [ ] 能在班级面前发言
- [ ] 能做 1-2 分钟的简短演讲
- [ ] 能克服基本的表达焦虑
- [ ] 能使用眼神接触
- [ ] 能组织简单的演讲结构
- [ ] 能回答听众的问题

### 创新思维

- [ ] 能想出多种解决方案
- [ ] 能把想法变成行动
- [ ] 能从失败中学习
- [ ] 能接受"不完美"的结果
- [ ] 能独立思考
- [ ] 能提出改进建议

### 消费决策

- [ ] 能理解广告策略
- [ ] 能为长期目标储蓄
- [ ] 能在预算内做选择
- [ ] 能区分"需要"和"想要"
- [ ] 能管理一周的零花钱
- [ ] 理解"劳动报酬"的概念

### 运动习惯

- [ ] 能广泛尝试不同运动
- [ ] 有运动偏好
- [ ] 每天户外活动 1 小时以上
- [ ] 能遵守运动规则
- [ ] 能接受输赢
- [ ] 喜欢运动而非被迫运动

### 视力保护

- [ ] 能遵守 20-20-20 法则
- [ ] 每天户外活动 1 小时以上
- [ ] 读写姿势正确
- [ ] 屏幕时间每天不超过 1 小时

### 睡眠管理

- [ ] 每天睡眠 9-12 小时
- [ ] 有固定的睡前例行活动
- [ ] 睡前 1 小时无屏幕
- [ ] 能独立入睡
