# 学龄中期（9-12岁）

## 阶段概述

9-12岁是具体运算期向形式运算期过渡的阶段。儿童开始具备抽象思维萌芽，自我调节学习能力增强，同伴群体认同需求上升。身体上进入专项运动启蒙和运动习惯锚定的关键窗口。

## 认知与心理主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 抽象思维萌芽 | 从具体向形式运算过渡 | 9-12岁 |
| 多步推理与假设生成 | 能够进行多步骤逻辑推理 | 9-12岁 |
| 批判性思维种子 | 质疑信息来源 | 9-12岁 |
| 自我调节学习 | 计划-监控-反思 | 9-12岁 |
| 信息素养 | 辨别信息真伪 | 9-12岁 |
| 同伴群体认同 | 归属感需求增强 | 9-12岁 |
| 抗逆力 | 面对挫折的恢复力 | 9-12岁 |
| 责任感与独立性 | 自主承担责任 | 9-12岁 |

## 身体能力主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 专项运动启蒙 | 尝试多种运动，避免过早单一化 | 9-12岁 |
| 速度与敏捷性发展 | 神经肌肉协调窗口 | 9-12岁 |
| 心肺耐力进阶 | 有氧能力提升 | 9-12岁 |
| 力量萌芽 | 神经系统适应为主，非肌肉肥大 | 9-12岁 |
| 运动习惯锚定 | 建立持久的运动习惯 | 9-12岁 |
| 运动损伤预防意识 | 正确热身、拉伸、恢复 | 9-12岁 |

## 理论依据

- 皮亚杰具体运算期成熟→形式运算萌芽
- Zimmerman自我调节学习模型
- Seligman习得性乐观与抗逆力
- Ryan & Deci自我决定理论（自主/胜任/归属）
- Selman观点采择阶段4（社会与习俗体系）
- LTAD学习训练阶段（Learn to Train）
- 运动采样期理论（Côté发展模型：Sampling阶段）
- 儿童抗阻训练安全指南（NSCA）
