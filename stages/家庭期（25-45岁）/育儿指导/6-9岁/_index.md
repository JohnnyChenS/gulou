# 学龄初期（6-9岁）

## 阶段概述

6-9岁是儿童进入正式学校教育的阶段。认知上进入具体运算期，开始掌握逻辑运算和守恒概念；社会性上从家庭中心转向同伴群体；身体上从基本动作向专项运动过渡。

## 认知与心理主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 守恒与逻辑运算 | 数量、质量、体积守恒 | 6-8岁 |
| 阅读习得 | 解码→流畅朗读→理解 | 6-9岁 |
| 元认知萌芽 | 开始知道"自己不知道" | 7-9岁 |
| 注意力持续增长 | 有意注意时间延长 | 6-9岁 |
| 规则理解与道德推理 | 从他律到自律过渡 | 6-9岁 |
| 友谊技能 | 互惠、忠诚、冲突解决 | 6-9岁 |
| 胜任感与自我效能 | 通过努力获得成功体验 | 6-9岁 |

## 身体能力主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 过渡性动作技能 | 从基本动作向专项运动过渡 | 6-9岁 |
| 游泳/骑行/球类入门 | 基础运动技能习得 | 6-9岁 |
| 柔韧性 | 自然柔韧期 | 6-9岁 |
| 正确体态养成 | 书写姿势、背包习惯 | 6-9岁 |
| 心肺耐力基础 | 有氧能力基础建设 | 6-9岁 |
| 精细动作 | 书写流畅性、键盘操作 | 6-9岁 |

## 理论依据

- 皮亚杰具体运算期（7-11）
- 埃里克森勤奋vs自卑（6-12）
- Kohlberg道德发展前习俗→习俗水平过渡
- Selman观点采择阶段3（相互观点采择）
- Bandura自我效能理论
- Flavell元认知发展理论
- LTAD学习训练阶段（FUNdamentals）
- Gallahue过渡性动作技能期
- 中国《国家学生体质健康标准》
