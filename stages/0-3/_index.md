# 婴幼儿期（0-3岁）

## 阶段概述

0-3岁是人生发展最迅速的时期，大脑神经连接以每秒百万级的速度形成。此阶段的核心任务是建立对世界的基本信任，发展感知运动能力，为后续所有发展奠定基础。

## 认知与心理主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 客体永久性 | 理解看不见的东西仍然存在 | 6-18月 |
| 因果关系推理 | 理解动作与结果之间的联系 | 4-12月 |
| 联合注意 | 与成人共享关注点 | 9-15月 |
| 符号思维萌芽 | 假装游戏、象征性使用物品 | 12-24月 |
| 安全依恋 | 信任世界的基本安全感 | 0-18月 |
| 自我意识觉醒 | 镜像识别，"我"的概念 | 15-24月 |
| 自主意志 | "我自己来"的独立意识 | 18-36月 |
| 共情萌芽 | 感知他人情绪 | 18-36月 |

## 身体能力主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 粗大动作链 | 俯卧抬头→翻身→独坐→爬行→扶站→独走→跑 | 0-24月 |
| 精细动作链 | 全掌抓握→拇指对捏→工具使用（勺/蜡笔） | 0-36月 |
| 感觉统合 | 前庭觉、本体觉、触觉的多通道协调 | 0-36月 |
| 口部运动 | 吸吮→咀嚼→发音器官协调 | 0-24月 |

## 理论依据

- 皮亚杰感知运动期（0-2）
- 埃里克森信任vs怀疑（0-1.5）、自主vs羞怯（1.5-3）
- Bowlby依恋理论
- Lewis镜像自我识别研究（15-24月）
- 蒙台梭利语言/感官/动作敏感期
- WHO运动发育里程碑标准
- Gallahue粗大动作发展模型
- Ayres感觉统合理论
