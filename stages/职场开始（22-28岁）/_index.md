# 职场开始（22-28岁）

## 阶段概述

职场开始是人生中从校园走向社会的关键过渡阶段，也是职业身份确立、财务能力提升、亲密关系深化的重要时期。此阶段的核心任务是在职场中建立专业能力，实现经济独立，同时在个人生活中建立稳定的亲密关系和独立生活能力。

## 学习路径图

```
专业知识体系深化（22-28岁）+ 职场社交网络经营（22-28岁）
  └→ 职业身份确立（22-28岁）
       └→ 工作-生活平衡（22-28岁）

财务素养进阶（22-28岁）→ 独立生活能力成熟（22-28岁）
亲密关系深化（22-28岁）→ 工作-生活平衡（22-28岁）
```
