# 成年期（25-40岁）

## 阶段概述

25-40岁是专业深耕和家庭建设并行的阶段。认知上从胜任走向精通，需要对抗思维固化；身体上生理机能从峰值开始缓慢下降，久坐生活方式成为主要健康威胁。

## 认知与心理主线

| 能力 | 说明 | 关键期 |
|------|------|--------|
| 专业精通 | 从胜任到专家 | 25-40岁 |
| 复杂问题解决 | 跨领域整合 | 25-40岁 |
| 指导他人 | mentoring能力 | 25-40岁 |
| 认知灵活性维持 | 对抗思维固化 | 25-40岁 |
| 领导力发展 | 影响力和组织能力 | 25-40岁 |
| 家庭经营 | 亲密关系维护、育儿 | 25-40岁 |
| 压力管理 | 职场与家庭双重压力 | 25-40岁 |
| 意义感与目标感 | 超越日常的目标追寻 | 25-40岁 |

## 身体能力主线

| 能力 | 说明 | 关键期 |

| 久坐对抗 | 办公室工作者的运动处方 | 25-40岁 |
