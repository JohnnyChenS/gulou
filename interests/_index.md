# 兴趣领域索引

本目录按兴趣领域组织跨年龄路径索引。每个兴趣目录下包含一个 `_index.md`，指向各年龄阶段对应的 Prompt 文件。

> Prompt 文件本身存放在 `stages/` 目录中，此处仅为索引导航。

## 如何贡献新兴趣领域

1. 在 `interests/` 下创建以兴趣名称命名的目录
2. 在目录下创建 `_index.md`，按年龄阶段列出对应的 Prompt 链接
3. 在 `stages/<阶段>/兴趣/<兴趣名称>/` 下创建实际 Prompt 文件
4. 提交 Pull Request
