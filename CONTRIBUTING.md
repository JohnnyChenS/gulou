# 贡献指南

感谢你对 AI 教育图谱项目的关注！本文档说明如何参与贡献。

## 贡献方式

### 1. 新增 Prompt

最核心的贡献方式。每个 Prompt 帮助一个家庭或学习者获得具体的能力培养指导。

**步骤**：
1. 确定目标阶段和能力主题（参考 `roadmap.md` 中的能力清单）
2. 在对应目录下创建 Prompt 文件（严格遵循格式规范，见 README.md）
3. 更新该阶段的 `_index.md`，在能力表格中添加条目
4. 如涉及兴趣副线，在 `interests/` 下创建或更新索引

### 2. 完善现有 Prompt

- 修正事实错误 → 标记 `fix`
- 补充引用文献 → 在文件和 `references/` 中同步更新
- 改进 Prompt 表述 → 保持模型无关性，不引入特定 AI 功能

### 3. 新增兴趣领域

1. 在 `interests/` 下创建以兴趣名称命名的目录
2. 创建 `_index.md`，按年龄阶段列出对应的 Prompt 链接（未覆盖的阶段标注"待贡献"）
3. 在 `stages/<阶段>/兴趣/<兴趣名称>/` 下创建实际 Prompt 文件
4. 更新 `interests/_index.md` 总览

### 4. 补充参考文献

- 在 `references/` 对应文件中添加 APA 格式引用
- 确保文献来源权威（同行评审期刊、权威机构指南、经典学术著作）
- 不可添加：自媒体文章、无出处转载、商业推广内容

## 审核流程

| 内容类型 | 审核要求 |
|---------|---------|
| 核心能力 Prompt | 需相关领域专家审核，标记为 `expert-reviewed` |
| 兴趣领域 Prompt | 至少一位社区成员评审，标记为 `community-contributed` |
| 文献引用 | 需确认来源权威性和引用格式 |
| 框架变更 | 需项目维护者审核 |

## 行为准则

- 尊重所有学习者 — 不因年龄、背景、能力差异而歧视
- 科学严谨 — 不编造理论或数据
- 保持中立 — 不在 Prompt 中植入商业推广或意识形态偏见
- 协作友善 — 对贡献者保持善意和包容

## 提交自检

提交 PR 前请确认：

- [ ] Prompt 文件遵循格式规范（frontmatter 字段完整、双模式 Prompt 均已编写）
- [ ] 引用文献在 `references/` 中有记录
- [ ] 对应的 `_index.md` 已更新
- [ ] `review_status` 已正确标注
- [ ] Prompt 内容模型无关、无商业推广
- [ ] Commit 消息遵循提交规范
